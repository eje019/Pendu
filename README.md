Notre version simplifiée du jeu du Pendu avec Java aura :

-Fenêtre d'accueil : avec un message de bienvenue, les règles du jeu, un bouton "Jouer" et un bouton "Quitter".

-Fenêtre de jeu : avec le mot secret masqué (ex: "_ _ _ _ _"), les lettres déjà proposées, le dessin ASCII du pendu qui évolue, et des boutons pour les lettres.

Pas de scores (on garde ça pour plus tard).

10 mots directement dans le code, dans un tableau.

Le pendu dessiné avec des caractères.
